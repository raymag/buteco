# Buteco
A simple buteco styled chat built with nodejs.

## Installing

### Clonning
```bash
$ git clone https://github.com/raymag/buteco.git
$ cd buteco
```

### Installing
```bash
$ npm install
```

### Running
```bash
$ npm src/server.js
```
